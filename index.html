Hand Aura Particles

Demo interaktif berbasis HTML yang bisa di-upload ke GitHub Pages. Gerakkan tangan ke kamera untuk mendorong, menarik, dan meledakkan partikel.

Menunggu izin kamera
Tips: buka lewat HTTPS atau localhost supaya izin kamera muncul normal.
Cara pakai:
1. Klik Aktifkan Kamera
2. Browser akan menampilkan popup izin kamera
3. Tekan Allow / Izinkan
4. Ujung telunjuk akan jadi kontrol partikel

Cubit jempol + telunjuk untuk efek ledakan.
